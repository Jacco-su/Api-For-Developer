获取课程分类 该接口用于获取课程分类

GET /api/jikexueyuan/type/

例：/api/jikexueyuan/type/

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": [
        {
            "type": "类型标示符",
            "title": "大类标题",
            "list": [
                {
                    "type": "类型标示符",
                    "title": "小类标题",
                    "list": [
                        {
                            "type": "类型标示符",
                            "title": "子类标题"
                        },
                    ]
                },
            ]
        },
    ]
}
{
    "error": 1,
    "msg": "错误信息"
}

获取课程列表 该接口用于获取课程列表

GET /api/jikexueyuan/courses/[type]/[page]

例：/api/jikexueyuan/courses/ios/1

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": [
        {
            "id": "课程编号",
            "title": "标题",
            "img": "封面图片"
        },
    ]
}
{
    "error": 1,
    "msg": "错误信息"
}

获取视频列表

GET /api/jikexueyuan/videos/[vid]

例：/api/jikexueyuan/videos/2837

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": [
        {
            "cid": "课程编号",
            "vid": "视频编号",
            "title": "标题"
        },
    ]
}
{
    "error": 1,
    "msg": "错误信息"
}

获取视频信息

GET /api/jikexueyuan/info/[cid]/[vid]

例：/api/jikexueyuan/info/2837/1

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": {
        "title": "标题",
        "url": "视频地址"
    }
}
{
    "error": 1,
    "msg": "错误信息"
}
