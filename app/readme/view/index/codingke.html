获取课程列表 该接口用于获取课程列表

GET /api/codingke/courses/[course]

course 可以取值的范围 { ios , android , swift , cocos2dx }

例：/api/codingke/courses/ios

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": [
        {
            "title": "阶段标题",
            "list": [
                {
                    "id": "cid课程编号",
                    "title": "课程标题"
                },
            ]
        },
    ]
}
{
    "error": 1,
    "msg": "错误信息"
}

获取视频列表

GET /api/codingke/videos/[vid]

例：/api/codingke/videos/206

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": [
        {
            "title": "章节标题",
            "list": [
                {
                    "id": "vid视频编号",
                    "title": "视频标题"
                },
            ]
        },
    ]
}
{
    "error": 1,
    "msg": "错误信息"
}

获取视频信息

GET /api/codingke/info/[cid]/[vid]

例：/api/codingke/info/206/1430

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": {
        "title": "视频标题",
        "url": "m3u8地址"
    }
}
{
    "error": 1,
    "msg": "错误信息"
}
