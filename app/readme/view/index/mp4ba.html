获取站点信息 该接口用于获取站点信息

GET /api/mp4ba/home

例：/api/mp4ba/home

  • 成功案例
  • 失败案例
{
    "error": 0,
    "title": "站点标题"
}
{
    "error": 1,
    "msg": "错误信息"
}

获取分类列表 该接口用于获取分类列表

GET /api/mp4ba/category

例：/api/mp4ba/category

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": [
        {
            "id": "类别id",
            "title": "类别名称"
        },
    ]
}
{
    "error": 1,
    "msg": "错误信息"
}

获取资源列表

GET /api/mp4ba/lists/[cid]/[page]

例：/api/mp4ba/lists/1/1

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": {
        "maxpage": "最大页码",
        "list": [
            {
                "id": "资源编号",
                "title": "资源标题"
            },
        ]
    }
}
{
    "error": 1,
    "msg": "错误信息"
}

获取资源信息

GET /api/mp4/item/[id]

例：/api/mp4ba/item/a3bf19ddc30842e2e27dfa18d53fc156f4dde0c7

  • 成功案例
  • 失败案例
{
    "error": 0,
    "data": {
        "title": "标题",
        "cover": "封面",
        "preview": [
            "预览图"
        ],
        "info": "简介",
        "resource": [
            "资源池"
        ]
    }
}
{
    "error": 1,
    "msg": "错误信息"
}
